THANKS FOR USING MY PROJECT
## HOW TO PLAY THE GAME:-
1- USE UP, DOWN, RIGHT & LEFT ARROW KEYS TO PLAY<br>
2- CLICK ON THE START BUTTON TO START<br>
3- CLICK ON THE END BUTTON TO EXIT 

## LIMITATIONS :
CAN NOT BE PLAYED DUAL
## REQUIREMENTS:
PYTHON 3
